## 信任资产

晚上和朋友吃饭，他分享了一个有趣的故事

他在推进一个项目，与 CEO 直接对接

在关键环节，卡住了，无法进一步往前推进

过了一段时间，这位 CEO 突然找他

说在抖音看到了一个 IP 的短视频，在介绍一篇论文，其中作者之一就是他

在得到肯定的答复后

CEO 说，咱们赶紧推进后面的协议，接下来就非常顺利了

这背后，反映了一个现状，对于很多合作，很多 CEO 并不是不想推进，而是缺少一个可以让自己放心拍板的理由

解决了信任、权威这种问题后，很多决策就容易很多了

这就是信任资产

信任有时候来自沟通，更多的，则是来自外部信号，比如权威人物的背书、你的作品、你的各种输出等

## 宝贵的常识

常识，是一个人在现实世界中反复有效的基础判断力

常识的价值和重要性，经常被忽略

常识，是人在信息不完整、时间有限、资源有限的真实环境里，用来判断事实、代价、风险、机会和行动后果的一套底层规则

10 条重要的常识：

1. 尊重事实，按事实运行，人的愿望不能替代事实

2. 机会成本，选择就是放弃，任何选择都有机会成本

3. 概率思维，概率思维比确定性思维更接近真实世界

4. 小概率大损失，比高概率小收益更值得优先处理

5. 激励决定行为，别只听人怎么说，要看他被什么奖励

6. 第一反应通常只看到眼前，高手会继续问：然后呢？做决定前，多问一句：然后会发生什么？

7. 复利思维，复利是世界上最朴素也最强大的增长规律

8. 少数关键因素，常常决定多数结果。世界的结果通常不是平均分布的。少数客户贡献多数收入，少数习惯决定大部分健康，少数关键决策决定人生走向

9. 先分清可控和不可控，否则会把力气浪费在错误地方

10. 真正的聪明，是保持谦逊、留有余地、持续修正

一个有常识、尊重常识的人，会尊重事实、计算代价、判断概率、重视长期，把力气用在关键且可控的事情上

## 步行冥想

今天走路去公司

公司所在的园区，在初夏天晴的时候，景色相当好，绿意葱葱，生机勃勃，人的状态也会跟着慢下来

在这样的地方，步行冥想，相当美妙

方法很简单：

1. 适当放慢走路速度

2. 把注意力放在脚下，以及身边具体的事物

3. 依次觉察身体的感受，脚踩在木制栈道的声音，身边拂过的凉风，摇曳的绿叶，还有各种鸟鸣

4. 走神了，不评判，只需再拉回来，继续感受

走路这件小事，如果加上冥想，就会从通勤变成一种“休息”与“惬意”

于是，给自己写了这个小工具，一键将长链接生成短链

一些说明：

1. 极简功能，输入长链接，自动生成短链和二维码，方便自己分享链接以及传播

2. 支持 Docker 部署，MIT 开源协议

3. 支持所有链接的访问数据统计以及用户管理

演示站点：[TokURL 演示站点](https://ai.laoyao.cn)

GitHub 地址：[TokURL](https://ai.laoyao.cn/TokURL)

底层模型是基于李叫兽的需求三角模型，效果不错

也就是，一个靠谱的需求，往往由 3 个核心要素组成，分别是：缺乏感、目标物和消费者能力

基本逻辑：

1. 输入一个产品，它会从需求三角、用户动机、支付意愿、竞争格局、风险成本等维度做诊断，并输出 HTML、Word、PDF 等多种报告

2. 核心解决产品的需求是否成立、哪里短板最大、下一步该验证什么

3. 无论是策划产品，或者某个功能，或者分析对标成熟产品，这个 Skill 会抓取相关数据，结合相关模型，给出详细的分析与评估

GitHub 地址：[需求评估 Skill](https://ai.laoyao.cn/Vnowb)
